# Personal Greeter

Плагин для VS Code, который приветствует пользователя по имени.

## Как работает
1. Пытается получить имя пользователя из Git конфигурации
2. Если Git не настроен, использует системное имя пользователя
3. Показывает приветствие при запуске VS Code

## Установка
1. Скопируйте папку в ~/.vscode/extensions/
2. Перезапустите VS Code
