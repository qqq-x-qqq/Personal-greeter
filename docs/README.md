# Personal Greeter - Документация

## Общее описание плагина

Personal Greeter - это плагин для Visual Studio Code, который приветствует пользователя по имени при каждом запуске редактора.

## Основные возможности

1. Автоматическая активация при старте VS Code
2. Определение имени пользователя из нескольких источников
3. Отображение персонализированного приветствия

### Функция `getUsername()`
Получает имя пользователя из доступных источников.

**Алгоритм работы:**
1. Пытается получить имя из конфигурации Git через `git config user.name`
2. Если Git не настроен, использует системное имя из `os.userInfo().username`
3. Если системное имя недоступно, возвращает "друг"

**Параметры:**
- Нет

**Возвращает:**
- `Promise<string>`: имя пользователя

**Пример:**
```javascript
const username = await getUsername();
// Возвращает "Иван" или имя системы
```

### Функция `activate()`
Основная функция активации расширения. Вызывается при запуске VS Code.

**Алгоритм работы:**
1. Получает имя пользователя через `getUsername()`
2. Отображает приветственное сообщение через `vscode.window.showInformationMessage()`
3. Формирует сообщение: `Привет, ${username}! Добро пожаловать в VS Code!`

**Возвращает:**
- `Promise<void>`

### Функция `deactivate()`
Функция очистки при деактивации расширения.

**Действие:**
- Выводит сообщение в консоль отладки
## Формат сообщений

- `Привет, qqq-x-qqq! Добро пожаловать в VS Code!` (имя из Git)
- `Привет, Ruslan! Добро пожаловать в VS Code!` (системное имя)
- `Привет, друг! Добро пожаловать в VS Code!` (значение по умолчанию)
